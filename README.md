# Help Center API Assignment

## Instructions

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/KumarRaj21/future_skills_fullstack_assignment
   ```

2. **Backend:**
   ```bash
     cd backend
     npm i express mongoose cors
     node index.js
   ```

3. **Frontend:**

    ```bash
    cd frontend
     npm i
     npm i axios react-icons
     npm start
   ```
